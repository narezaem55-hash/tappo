# Tappo Web (лендинг)

## Запуск локально
1) Установи Node.js (LTS)
2) В папке проекта:
```bash
npm install
npm run dev
```
Открой http://localhost:3000

## Деплой на Vercel (самый простой)
1) Создай аккаунт на Vercel
2) Нажми "New Project" → "Import"
3) Загрузить можно через GitHub (рекомендуется) или "Upload"
4) После деплоя подключи домен в Settings → Domains
